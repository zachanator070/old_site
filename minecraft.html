How to play the ZachPack

So I hear you want to play minecraft on this server? Well there are a couple of things that we need to address before you can play on thezachcave.com.

Windows

Grats. Things will be a little easier for you. Things you will need:

  • The forge mod loader: forgemodloader-1.6.4
  • The installer for the zachpack: ZachPack.bat

Now follow these steps and you are ready to go!

  1. Run the Forge Mod Loader installer and install the "Client" version, then walk through the rest of the launcher
  2. If you have never played Minecraft ver 1.6.4 then click on edit profile and change your version to 1.6.4 and run the game. After running the game quit and run the minecraft launcher again. Screenshot here
  3. Run the ZachPack.bat file which will download and extract the config and mod files into your minecraft install directory
  4. Change your profile to the "Forge" profile and click play. Screenshot here
  5. Click multiplayer and add the server thezachcave with the address thezachcave.com

Manual Install

If the script isnt working then you will have to download the files and extract them yourself

  • The forge mod loader: forgemodloader-1.6.4
  • The mods for the zachpack: mods.rar
  • The config files for the mods: config.rar

After you grab all these files you will need to complete the following steps in this order:

  1. Run the Forge Mod Loader installer and install the "Client" version, then walk through the rest of the launcher
  2. Extract the files from mods.zip and put them into your minecraft directory in the mods folder. Screenshot here
  3. Extract the files from config.zip and put them into your minecraft directory in the config folder
  4. If you have never played Minecraft ver 1.6.4 then click on edit profile and change your version to 1.6.4 and run the game. After running the game quit and run the minecraft launcher again. Screenshot here
  5. Change your profile to the "Forge" profile and click play. Screenshot here
  6. Click multiplayer and add the server thezachcave with the address thezachcave.com

Notes

  • *NEW* The supported texture pack can be found here featuring the Sphax Pure BDcraft textures! Just put the zip file in your minecraft directory in the resources folder and select the texturepack when you run the game to apply the textures.
  • If you want to switch back to vanilla minecraft just change your profile back to your username the next time you launch minecraft.
  • If you get any errors about id conflicts be sure to download and apply the config files
  • If you have an error be sure you have followed the steps and then tell me about it
